IMAGES: how to drop in your real photos
========================================

The site is set up so you can add real photos with ZERO code changes.

Each image on the site first tries to load a real photo (a .jpg). If that file
is not here yet, it automatically falls back to a clearly labelled .svg
placeholder, and a small "Placeholder image" chip shows in the corner. The
moment you add the real .jpg with the right name, the photo appears and the
chip disappears on its own.

WHERE TO GET PHOTOS
-------------------
Free, no-attribution stock is fine for a demo:
  - Pexels:  https://www.pexels.com/search/australian%20native%20plants/
  - Pixabay: https://pixabay.com/images/search/native%20plants/
Pick warm, natural photos of native plants, hands, soil and gardens.

WHAT TO NAME THEM (save into this folder, assets/images/)
---------------------------------------------------------
Every file is prefixed with "sfc-" (Seeds for Country) so it never clashes with
images from any other website you build. Use these EXACT names:

  sfc-hero-hands-seedlings.jpg   Home hero. Hands cupping native seedlings or seed.
  sfc-about-seeds-soil.jpg       About section. Close-up of seeds and rich soil.
  sfc-nursery-partner.jpg        Partner. An Aboriginal-owned native plant nursery.
  sfc-restoration-partner.jpg    Partner. Aboriginal-led on-Country restoration.
  sfc-product-pack.jpg           Product page. A flat-lay of the seed pack and guide.

Use .jpg and those exact names. If you only have .png, just rename it to .jpg,
or tell me and I will switch the slot over.

IMPORTANT for the two partner photos (sfc-nursery-partner.jpg,
sfc-restoration-partner.jpg): these should be provided and credited by the
Aboriginal partner, not chosen on their behalf. Update the alt text and add a
credit line when you add them.
